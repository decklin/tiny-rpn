Tiny RPN

Tiny RPN is a Reverse Polish Notation calculator, operated entirely with the keyboard. It displays the contents of the stack as you modify it (WYSIWYC â€” What You See Is What You Calculate :-)).

Instructions

  • To enter several numbers, separate them with spaces or commas, or press Enter/Return after each one. Negative numbers can be entered by using a leading underscore.

  • To perform an arithmetic operation, type its character (e.g. + for addition) in the entry box; instead of displaying the character, the stack will be updated. The arithmetic operations are:

    + - * + / % ~ ^ & | \

    (^ is exponentiation, because most people expect it there, so xor is moved to \. ~ is unary negation.)

  • To perform a named operation, type in its name followed by a space/comma/enter. Named commands are:

    and or not pop drop drop2 swap dup dup2 clear noop

    plus these standard (to JavaScript, anyway) math functions:

    abs acos asin atan atan2 ceil cos exp floor log max min pow random round sin sqrt tan

    and these constants:

    E LN2 LN10 LOG2E LOG10E PI SQRT1_2 SQRT2

    (Also, at the moment, ; is an alias for swap. All such shortcut keys will be subject to change until I get some consensus from users on what is sensible.)

  • Numbers can also be popped by pressing the Backspace key when the entry box is empty. If you use this method or pop, the number on the top of the stack will be returned to the entry box so you can edit it. drop will remove the number on top of the stack without doing this.

  • The popup can be closed with the Esc key (this is true of all browser action popups, but I didn't realize it until I wrote one that involved typing).
