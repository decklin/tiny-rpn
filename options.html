Options

(Looking for the help page?)

Define any custom operators here, by adding attributes to the already-defined object ops. They should be functions following these rules:

  1. Take the number of arguments that you want popped from the stack. They will be applied to you in bottom-up order.

  2. If you are pure, return a value or (possibly empty) list of values you want pushed back onto the stack (again, bottom-up). The resulting head will be pushed onto the undo list.

  3. If you have side effects, don't return anything, just create a new head (nothing will be pushed onto the undo list above it).

Single-character operator names will dispatch as soon as their key is pressed. Be careful! I recommend only using punctuation for this.

To define a new control-key binding, add an attribute (which must be an uppercase letter) to ctrlBindings.

You can also change the default values of inputRadix, outputRadix, or termSep (a regex).
